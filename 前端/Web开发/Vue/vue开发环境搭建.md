## 2. Vscode

推荐使用的 IDE 是 VSCode，配合 Vue 语言特性 (Volar) 插件。该插件提供了语法高亮、TypeScript 支持，以及模板内表达式与组件 props 的智能提示。


## 2.  vite

Vite 是一种新型前端构建工具，能够显著提升前端开发体验；Vue官方建议使用Vite作为Vue项目的构建工具。

[vite](https://cn.vitejs.dev/)

```sh
npm create vite@latest
```
