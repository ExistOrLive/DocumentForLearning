# 语法

## 注释

> python 使用`#`,`'''`以及`"""`来添加注释


```python

# 第一个注释

'''
第二个注释
第三个注释
'''

"""
第四个注释
第五个注释
"""

```

### 行与缩进

> python使用缩进来表示代码块，不需要使用{}；缩进的空格数是可变的，但是同一个代码块的语句必须包含相同的缩进空格数。

```Python

# 条件语句
if True:
    print("True")
else:
    print("false")

# 循环语句
for i in array:
    print(i)

if a > 1:
    print("a>1")
elif a < 10:
    print("a<10")
else:
    
    
```
