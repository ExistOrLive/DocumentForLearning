# bool/BOOL/Boolean

- BOOL 
   
      实际上是unsigned char。当接受数值类型时，只会接受低8位。

      YES==1 NO==0 

- Boolean 
        
      typedef unsigned char                   Boolean;

      实际上是unsigned char。当接受数值类型时，只会接受低8位。

- bool 
      
      bool类型的变量只有两个值，true和false

      接受非0值时，bool就是true；接受0值时，bool就是false


Tip：

BOOL｜Boolean｜bool 都不要和数值类型之间直接进行互相赋值。
