# 概述
HTML5 不是简单的HTML标记语言的新版本，而是包括 HTML，CSS 和 JavsScript API 整体的更新版本，将Web做为应用开发平台，并制定了Web应用开发的一系列标准。

##### HTML
- 语义化更近一步的语义化
- 提供了更多的标签和属性，更多的操作，增强了交互性
- 支持多媒体 `<video> <audio>`
- 支持自定义标签
- 支持绘制2D/3D图形  `<canvas>`
