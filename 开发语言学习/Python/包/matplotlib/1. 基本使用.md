# 基本使用

```py
import matplotlib as mpl
import matplotlib.pyplot as plt
import numpy as np
```

## 创建图表

- `pyplot.subplots`创建图表

- `Axes.plot`在轴上绘制数据

```py
# subplots 创建图表和轴
# fig 图表
# ax 子轴
fig, ax = plt.subplots()
ax.plot([1,2,3,4],[1,4,2,3])
plt.show()
```
