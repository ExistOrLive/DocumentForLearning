### 1. 算术运算


`/` 运算符返回结果为 浮点数 ， `//` 返回结果为 整数
​

`**`运算符用于 乘方 运算
​

```python
>>> 5 + 4  # 加法
9
>>> 4.3 - 2 # 减法
2.3
>>> 3 * 7  # 乘法
21
>>> 2 / 4  # 除法，得到一个浮点数
0.5
>>> 2 // 4 # 除法，得到一个整数
0
>>> 17 % 3 # 取余
2
>>> 2 ** 5 # 乘方
32
```

### 5. 成员运算符 


`in` `not in` 运算符 检查某个元素是否在集合中 (arrray, set, string)
​

```python
a = 10 
b = 11
array = [1,2,10]

print(a in array)

print(b not in array)
```
