# SwiftUI工程

## 1. 创建SwiftUI工程

创建一个iOS App， **Interface** 选择 `SwiftUI`, **Life Cycle** 选择 `SwiftUI App`, **Language** 选择 `Swift`.


`SwiftUI` 是 不同于 `UIKit` 的 UI 框架， 因此在创建工程时，不再使用 `UIKit` 和 `StoryBoard`。
