# UNIX标准及实现

## UNIX标准化

### ISO C
> **ISO C** 是国际标准化组织（ISO）开发并维护的C语言标准，提高C语言的可移植性，使其适用于各种操作系统；现在UNIX，Linux，Windows都已基本支持ISO C

> 现在常用的ISO C版本 C99 C11

ISO C 库的头文件可以分为24个区，POSIX.1标准包括这些头文件以及另外一些头文件。
```
<assert.h>       ----------------------   验证程序断言
<complex.h>    ----------------------   支持复数算术运算
<ctype.h>        ----------------------   字符类型
<errno.h>        ----------------------   出错码
<fenv.h>          ----------------------   浮点环境
<float.h>          ----------------------   浮点常量
<inttypes.h>    ----------------------   整型格式转换
<iso646.h>      ----------------------   替代关系操作符宏
<limits.h>         ----------------------   实现常量
<locale.h>       ----------------------   局部类别
<math.h>         ----------------------   数学常量
<setjmp.h>      ----------------------   非局部goto
<signal.h>       ----------------------   信号
<stdarg.h>      ----------------------   可变参数表
<stdbool.h>     ----------------------   布尔类型和值
<stddef.h>       ----------------------   标准定义
<stdint.h>        ----------------------   整型
<stdio.h>         ----------------------   标准I/O库
<stdlib.h>        ----------------------   实用程序库函数
<string.h>       ----------------------   字符串操作
<tgmath.h>     ----------------------   通用类型数学宏

<wchar.h>      ----------------------   扩展的多字节和宽字符支持
<wctype.h>     ----------------------   宽字符分类和映射支持
 
```
